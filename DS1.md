## Invariante de Loop

Uma invariante de loop é uma asserção lógica, uma propriedade do loop de software que é verdadeira antes, e depois, de cada iteração.

### Coisas:

- Lista
- Fila
- Pilha
- Busca
- Ordenação
- N^2 e nlogn
- Merge sort, bubble sort
- Análise e prova:
- Theta(n)
- Invariante de loop
- Indução
- Recursão
